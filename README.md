# My Portfolio

Personal portfolio website showcasing projects, certifications, and skills.

## Live Demo

- GitHub Pages: https://abhi0626-kr.github.io/My_Portfolio/

## Project Structure

- `index.html` (root): Redirect entry for GitHub Pages
- `My_Portfolio/index.html`: Main portfolio page
- `My_Portfolio/style.css`: Styles
- `My_Portfolio/script.js`: JavaScript interactions
- `My_Portfolio/Badge/`: Badge and certification images
- `My_Portfolio/*.jpg|*.png|*.jpeg`: Portfolio assets

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/abhi0626-kr/My_Portfolio.git
   ```
2. Open the folder in VS Code.
3. Open `My_Portfolio/index.html` in your browser.

## Deployment (GitHub Pages)

This repo is configured to deploy from the root branch.
The root `index.html` redirects to `My_Portfolio/index.html` so Pages can find a default entry file.
